🔢 num-format-js

Blazingly fast number formatting with WebAssembly

🦀 Rust + WASM ⚡ High Performance 🌍 Locale Support 🎨 Custom Formatting

🌍 Locale Examples

Format numbers according to different locales with a single function call. Each country has its own formatting rules for thousands separators and decimal points. 

import { format_string } from '@rprojects/num-format-js';

// Format with any locale
const result = format_string('en', 1234567);
console.log(result); // "1,234,567"

🎨 Custom Formatting

Take full control with custom separators, decimal points, and signs. All settings are optional - use defaults or override what you need.

🎯 Custom Format Demo

Formatted Result
import { format_with_custom_locale } from '@rprojects/num-format-js';

// All settings are optional!
const result = format_with_custom_locale(-9876543, {
  grouping: 0,          // 0: Standard, 1: Indian, 2: Posix
  minus_sign: '−',
  plus_sign: '+',
  decimal: ',',
  separator: ' '
});
